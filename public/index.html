Setup Requirements

To get started, you'll need to configure the following:

  1. OpenAI API Key: Get your API key from OpenAI Platform and add it to your .env file
  2. Twilio Account: Sign up at Twilio and purchase a phone number with Voice capabilities
  3. ngrok Setup: Download ngrok and run ngrok http 5050 to expose your local server
  4. Webhook Configuration: In Twilio Console, set your phone number's webhook to https://your-ngrok-url.ngrok.app/incoming-call

Need help? Check the documentation for detailed setup instructions.
